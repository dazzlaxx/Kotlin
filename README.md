# GitHub Repository Report Generator

Консольное приложение на Kotlin для генерации CSV-отчетов о репозиториях пользователей GitHub. Приложение получает данные через GitHub REST API, вычисляет агрегированные показатели и сохраняет структурированный отчет в формате CSV.

## Функциональность

- Получение списка репозиториев заданного пользователя GitHub
- Извлечение ключевых метрик: звезды, форки, язык программирования, дата обновления
- Расчет агрегированных показателей:
    - Общее и среднее количество звезд
    - Общее количество форков
    - Статистика по языкам программирования
    - Определение самого популярного репозитория
- Генерация CSV-отчета, совместимого с Excel и другими табличными редакторами
- Обработка ошибок (пользователь не найден, превышение лимита API, сетевые ошибки)

## Технологии

- **Язык:** Kotlin 2.1.10+
- **Сборка:** Maven
- **HTTP-клиент:** OkHttp 4.12.0
- **JSON-парсинг:** Jackson 2.18.2 (модуль для Kotlin)
- **API:** GitHub REST API v3

## Требования

- JDK 17 или новее
- Maven 3.6+
- IntelliJ IDEA (рекомендуется) или другая IDE с поддержкой Kotlin
- Доступ к интернету для выполнения запросов к GitHub API
